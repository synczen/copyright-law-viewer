# 저작권법 3단 비교 서비스

저작권법과 시행령, 시행규칙을 한눈에 비교하고 조문별로 쉽게 탐색할 수 있는 법률 정보 웹사이트입니다.

## 주요 기능

### 1. 3단 비교 뷰
법률, 시행령, 시행규칙을 가로로 나열하여 동시에 비교할 수 있습니다. 각 문서 유형은 색상으로 구분됩니다:

- **법(파란색)**: 저작권법의 기본 조문
- **시행령(주황색)**: 법에서 위임한 구체적인 사항
- **시행규칙(초록색)**: 법 및 시행령의 시행에 필요한 세부 사항

### 2. 인터랙티브 네비게이션
좌측 패널에서 다음 기능을 제공합니다:

- **법률 선택 탭**: 저작권법, 시행령, 시행규칙 중 원하는 법률 선택
- **챕터별 아코디언**: 제1장 총칙부터 제13장 벌칙까지 장별로 조문 구성
- **조문 목록**: 각 장 내의 모든 조문을 번호와 제목으로 표시

### 3. 실시간 검색
상단 검색 필드에서 조문 번호 또는 제목으로 빠르게 검색할 수 있습니다. 검색 결과는 드롭다운으로 표시되며, 클릭하면 즉시 해당 조문으로 이동합니다.

### 4. 법률 추가 기능
"법률 추가" 버튼을 클릭하여 새로운 법률을 추가할 수 있습니다. 추가된 법률은 로컬 스토리지에 저장되며, 향후 조문 데이터를 입력할 수 있습니다.

## 사용 방법

### 조문 탐색
1. 좌측 패널에서 원하는 법률을 선택합니다.
2. 장을 클릭하여 펼치고 조문을 선택합니다.
3. 중앙 패널에 선택한 조문의 3단 비교 뷰가 표시됩니다.

### 검색으로 조문 찾기
1. 상단 검색 필드에 조문 번호(예: "제10조") 또는 제목(예: "저작권")을 입력합니다.
2. 검색 결과가 드롭다운으로 표시됩니다.
3. 원하는 조문을 클릭하면 해당 조문으로 이동합니다.

### 새로운 법률 추가
1. "법률 추가" 버튼을 클릭합니다.
2. 법률명(한글, 영문), 설명, 제정일을 입력합니다.
3. "추가" 버튼을 클릭하면 새로운 법률이 추가됩니다.

## 데이터 구조

### laws.json
모든 법률 데이터는 `/client/public/laws.json` 파일에 저장됩니다. 구조는 다음과 같습니다:

```typescript
{
  "laws": [
    {
      "id": "copyright_law",
      "name": "저작권법",
      "englishName": "Copyright Act",
      "lawType": "law",
      "enactDate": "1957-01-28",
      "lastModified": "2025-09-26",
      "description": "...",
      "relatedLaws": ["copyright_decree", "copyright_rule"],
      "chapters": [...],
      "articles": [...]
    }
  ],
  "tripleComparisons": [...],
  "lastUpdated": "2025-09-26"
}
```

## 설계 철학: "Judicial Clarity"

이 웹사이트는 다음 원칙을 따릅니다:

- **신뢰성 우선**: 깔끔하고 전문적인 디자인으로 신뢰감 구축
- **정보 계층화**: 법-령-규칙의 3단 비교를 명확하게 구분
- **스캔 가능성**: 사용자가 빠르게 원하는 조문을 찾을 수 있는 명확한 네비게이션
- **접근성**: 모든 법률 사용자(법률가, 학생, 일반인)를 위한 직관적 인터페이스

### 색상 팔레트
- **기본색**: 깊은 남색(#1e3a5f) - 신뢰감과 전문성
- **강조색**: 따뜻한 골드(#d4a574) - 중요 정보 강조
- **배경색**: 순백색(#ffffff) - 텍스트 가독성 극대화
